# Innowise Solo Leveling. Foursquare

REST-API practice on Android in Kotlin.
## Features

- Room for storing data
- Retrofit requests
- Filtering
- OAuth 2.0
- Picasso for loading images
- Colorfull icons from API icons
- Place tips
- Image slider
- Splash screen
- Data binding
- Live Data
